# 更新日志

## [1.1.0] - 2026-03-31

### 升级 - 智能推荐引擎 v2

#### 第一期：基础增强
- **多阶段关联评分系统**
  - 替代原有独立打分模式，采用序列化选品策略
  - 上衣→下装→外套→鞋→包，每步都考虑已选单品的协调度
  - 五维评分：颜色和谐(30%) + 风格一致(25%) + 正式度协调(20%) + 场合匹配(15%) + 用户偏好(10%)
  - 整套搭配最终评分，展示协调度百分比

- **数据驱动风格矩阵（替代硬编码）**
  - 场合×风格的二维评分矩阵（10场合×10风格 = 100个可调权重）
  - 正式度等级量化（0.0最休闲 ~ 1.0最正式）
  - 颜色搭配评分矩阵（36组经典配色对，3级评分）
  - 支持学习系统动态修改权重

- **接入 Open-Meteo 真实天气 API**
  - 免费、无需 API Key、支持全球城市
  - 基于 CoreLocation 自动获取用户位置
  - WMO 天气编码映射（11种天气状况）
  - 网络不可用时自动降级为模拟数据
  - 新增雷暴、小雨天气类型，防雨提醒

#### 第二期：智能化
- **统计学习服务 LearningService**
  - 穿搭日记评分 → 强信号（±0.03/次）
  - 保存推荐方案 → 正向信号（+0.02/次）
  - 刷新推荐 → 弱负向信号（-0.005/次）
  - 衣物穿着频率 → 隐式偏好（+0.01/次）
  - 动态调整颜色组合、风格、品类权重
  - 自动同步至风格矩阵

- **Apple Foundation Models 接入（iOS 26+）**
  - 设备端大模型，完全离线、隐私安全
  - @available 兼容，iOS 17-25 自动降级为规则引擎
  - 规则引擎支持 6 类智能回复（搭配/天气/配色/购物/点评/风格）

- **AI 穿搭助手聊天界面**
  - 全新对话式交互，自然语言问答
  - 5个快捷操作按钮
  - 气泡式消息布局，支持多轮对话
  - 实时上下文感知（衣橱/天气/用户偏好）

### 技术
- 新增 5 个服务文件、2 个视图文件
- LocationManager 定位服务
- Tab 栏新增「助手」入口
- ViewModel 全面接入学习信号采集
