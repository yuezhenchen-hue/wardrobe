# 更新日志

## [1.0.0] - 2026-03-31

### 新增
- **衣橱管理模块**
  - 衣物添加（拍照/相册选择）
  - 衣物分类（上衣、下装、外套、连衣裙、鞋子、包包、配饰）
  - 衣物属性标签（颜色、材质、季节、风格、保暖程度）
  - 衣物搜索、筛选、排序
  - 衣物收藏功能
  - 衣物详情查看（含配色建议）
  - 穿着次数与日期记录

- **智能穿搭推荐模块**
  - 基于天气的穿搭推荐
  - 基于场合的穿搭推荐（10种场合）
  - 配色和谐度分析评分
  - 多套方案生成
  - 推荐方案保存

- **穿搭日记模块**
  - 每日穿搭记录
  - 月历视图标记
  - 心情记录（8种心情）
  - 穿搭评分（5星制）
  - 连续打卡统计
  - 风格趋势分析

- **个人形象模块**
  - 用户资料管理（身高、体重、体型、肤色）
  - 风格偏好测试（5题趣味测试）
  - 基于肤色的颜色推荐
  - 头像上传

- **购物建议模块**
  - 衣橱分类统计分析
  - 缺失单品智能推荐
  - 购物小贴士

- **通用功能**
  - 新用户引导流程
  - 天气显示横幅
  - 数据本地持久化（UserDefaults + JSON）
  - 现代化 UI 设计（卡片式、渐变色、动画）
  - 示例数据预置

### 技术
- Swift 5.9 + SwiftUI
- MVVM 架构
- iOS 17.0 最低支持
- XcodeGen 项目管理
